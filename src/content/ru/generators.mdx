# Generators in animation

Разберём как сделать сложную анимацию компонента используя генераторы в джс и реакт.

Для примера возьмём простой компонент – лоадер, который должен показывать прогресс и менять контент по мере заполнения.

Далее появляется задача сделать прерывание анимации для того, чтобы пользователь что-то выбрал, например промпт.

Сделать это стандартными средствами реакта достаточно сложно. Поэтому на помощь приходит оркестрирование с помощью генераторов.

Для начала нужно разобраться как работают генераторы.
